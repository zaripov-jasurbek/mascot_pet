  Маскот — голосовой настольный помощник
═══════════════════════════════════════════════

КАК ЗАПУСТИТЬ
  1. Распакуйте эту папку целиком (не запускайте из архива!).
  2. Дважды кликните  mascot.exe
  3. Подождите пару секунд — на экране появится персонаж.

КАК ПОЛЬЗОВАТЬСЯ ГОЛОСОМ
  Скажите в микрофон, начиная со слова «помощник»:

    «помощник открой телеграм»            — запустит приложение
    «помощник открой ютуб в браузере»     — откроет сайт
    «помощник открой авито в браузере»    — найдёт сайт через поиск

  Без слова «помощник» команды игнорируются (чтобы не срабатывать
  на обычную речь и фоновый звук).

НАСТРОЙКА (необязательно)
  Файл voice.toml можно открыть Блокнотом и поменять:
    wake   — слово-активатор (по умолчанию «помощник»)
    [site] — свои быстрые ссылки на сайты
  После изменений — перезапустите mascot.exe.

ЕСЛИ НЕ ЗАПУСКАЕТСЯ
  • Нужна видеокарта с поддержкой Vulkan (почти все современные ПК).
  • Windows может предупредить о неизвестном издателе — «Подробнее» →
    «Выполнить в любом случае».
  • Не работает микрофон? Проверьте, что он выбран в настройках звука
    Windows и приложению разрешён доступ к микрофону.
