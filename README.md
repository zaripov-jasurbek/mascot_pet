# Маскот — голосовой настольный помощник

Анимированный 3D-персонаж (VRM) на рабочем столе с оффлайн-распознаванием речи.
Скажи «помощник открой …» — запустит приложение или откроет сайт. Всё локально,
без интернета и облака.

---

## Для пользователей (просто запустить)

Не нужен ни Rust, ни сборка.

1. Открой страницу **[Releases](../../releases)** этого репозитория.
2. Скачай `mascot-win64.zip` из последнего релиза.
3. Распакуй папку и запусти **`mascot.exe`** (двойной клик).

Подробности — в файле `ПРОЧТИ-МЕНЯ.txt` внутри архива.

> Требуется Windows 10/11 с видеокартой, поддерживающей Vulkan (почти все
> современные ПК), и микрофон.

---

## Для разработчиков (собрать из исходников)

### 1. Зависимости
- [Rust](https://rustup.rs) (toolchain MSVC) + Visual Studio Build Tools (C-компилятор для `rusqlite`).
- Нативная библиотека `libvosk` и модель распознавания — **не в репозитории**
  (большие бинарники, в `.gitignore`). Скачиваются скриптом:

```powershell
powershell -ExecutionPolicy Bypass -File scripts\fetch-deps.ps1
```

Скрипт положит `libvosk` в `vendor\` и русскую модель в `models\`.

### 2. Запуск
```powershell
$env:PATH = "$PWD\vendor\vosk-win64-0.3.45;$env:PATH"   # чтобы нашёлся libvosk.dll
cargo run
```

### 3. Сборка готового пакета для раздачи
```powershell
powershell -ExecutionPolicy Bypass -File scripts\package.ps1
```
Создаст `dist\mascot\` (exe + DLL + модель + assets + voice.toml) и
`dist\mascot-win64.zip` — этот zip и публикуется в Releases.

### 4. Авто-релиз (GitHub Actions)
Можно не собирать вручную: запушь тег — CI сам соберёт zip и опубликует Release.
```powershell
git tag v0.1.0
git push origin v0.1.0
```
Workflow: `.github/workflows/release.yml` (ручной запуск через *Actions → Release →
Run workflow* кладёт zip в Artifacts без публикации).

## Голосовые команды

| Скажи | Что произойдёт |
|---|---|
| `помощник открой телеграм` | запуск приложения (поиск по меню «Пуск») |
| `помощник открой ютуб в браузере` | открыть сайт |
| `помощник открой <любой сайт> в браузере` | поиск сайта (история браузера → Google) |

Команда срабатывает только после слова-активатора (по умолчанию «помощник») —
защита от ложных срабатываний на фоновую речь.

## Настройка — `voice.toml`

Читается при запуске (менять без пересборки): модель/язык, слово-активатор,
глаголы, поисковик, быстрые ссылки `[site]`. Пути приложений **не** хранятся —
индексируются автоматически из меню «Пуск» и `PATH`.
